# Contributing to Balance

Thanks for your interest in contributing. This repo contains a React Native + Expo app.

## Quick start
1) Install dependencies
   - `npm install`
2) Start the app
   - `npm run start`
   - iOS: `npm run ios`
   - Android: `npm run android`
   - Web: `npm run web`

## How to contribute
- Keep changes small and focused.
- Follow existing patterns and UI tokens.
- Avoid adding new dependencies unless necessary.
- Update i18n strings in all locales (en/it/pt) when touching user-facing copy.
- Do not add or log sensitive user data.

## Tests
There are a few tests under `__tests__/`.
If you add or change tests, run:
- `npx jest`

## Pull request checklist
- Clear title and short description of what changed.
- Screenshots or short video for UI changes.
- Note any manual testing you performed.
